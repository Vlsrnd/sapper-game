### Install
```
npm install
```
### Usage
```
npm run start
```
### Build
```
npm run build
```

What was used:
- Webpack 5;
- JS:
  + object, array, Map, Class;
  + destructuring assignment;
  + JSON;
  + scheduling: setTimeout;
  + closure;
  + spread;
  + async/await;
  + work with DOM;
  + matchMedia (media queries from JS);
  + browser event, event delegation;
  + mouse event;
  + forms;
  + localStorage;
  + canvas (2d);
  + requestAnimationFrame;
